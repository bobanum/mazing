# Mazing
![](assets/logo_mazing.svg)
Every maze is a puzzle, but not every puzzle is a maze.

## Description
Mazing is a Javascript maze generator, solver and renderer.

## Features
- Generate a maze
- Solve a maze
- Render a maze
